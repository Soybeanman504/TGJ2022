「ゆっくりできない100秒間」
URL:https://soybeanman504.github.io/TGJ2022/

=====

・これは何？
東方ゲームジャム2022に投稿した東方Projectの二次創作ゲームです。
ゆっくりを撫でて（そして消して……そんな……）点数を稼ぐ爽快パズルゲームです。
100秒の時間制限があり、その間に点数をできるだけ稼ぐのが目標となります。

・テーマは？
全てです。
「紅」→紅魔郷主人公組・ボスのゆっくりが使用できる。
「つなぐ」→ゆっくりをつないで得点を稼ぐ。
「100」→100秒の時間制限。
「ゆっくり」→ゆっくりを撫でられる（そして消せる）。

・対象機器
ブラウザでインターネットにつなげられるPCやスマホ

・操作方法
クリックorタップで操作します。

=====

・タイトル画面
最初の画面です。
STARTを押すとゆっくりセレクト画面に移ります。

・ゆっくりセレクト画面
使用するゆっくりを4人選びます。
初期設定では霊夢・魔理沙・ルーミア・チルノが選択されています。
ゆっくりの下に表示されている「点:P=」の表示は消した時の得点を点数と霊力(P)に分配する際の割合を表しています。
STARTを押すとゲーム画面に移ります。

・ゲーム画面
3,2,1,STARTの合図の後にゲームが始まります。
左上の数字が制限時間、右上が点数、霊力(P)、右下がコンボ数、下がスペルカード使用ボタンです。

2つ以上の同じ種類のゆっくりを左クリック長押し・ドラッグで選択すると（撫でると）消えます。
撫でたゆっくりの数が多いほど高得点になります。

前回消してから2秒が経たないうちにゆっくりを消すとコンボが繋がります。
コンボ数が10までの間はコンボ数に比例して追加点数が上がっていきます。

スペルカードは霊力(P)が1以上になると使用できます。
下のスペルカード使用ボタンを押すと発動します。
効果は使ってみてからのお楽しみ♡

100秒経つとリザルト画面に映ります。

・リザルト画面
結果が確認できます。
Twitterアカウントをお持ちの方は点数をツイートして自慢することができます。
ぜひフォロワーと競ってみてください！
RESTARTを押すともう一度ゲームができます。
